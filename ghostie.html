Ghostie

PERSONAL

DEVELOPMENT

GAME JAM

During university I collaborated with a friend on a series of game jam projects. We used the game development software Godot and wrote our code in GDScript, which has a similar syntax to Python, but allows easier integration of built in Godot tools such as physics engines.

Ghostie was our first attempt at a game jam and game development as a whole so it is slightly rough around the edges! The theme of the game jam was 'combining two genres', and after planning and discussing we landed on a mix of platformer and puzzle game.

Then over the next day and a half we designed and coded Ghostie from the ground up, learning any GDScript specific syntax as we went. For a first attempt and the limited time frame I'm pretty happy with the result.

You can play the game here: https://taylorjones.itch.io/ghostie2
